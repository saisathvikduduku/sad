The Django app is developed by python. So the requerements for Django app development is JsonRequest, python, virtualenv, and web browser. 
step 1
	On the virtual environment, django package is required to be installed. 
step 2
 	Create Django app by command of "django-admin startproject helloworld"

step 3
	Run the manage.py which is created in the project of helloworld 
step 4
	Create a python file with data to printing the message of helloworld" by Jsonrequests.

step 5
	In the urls file, the path is setting up with the newly created helloworld program. 

step 6
	Run the manage.py file and get the url for running in virtual browser.

step 7

	Using the url IP address loading on browser, hello world message is printed.
